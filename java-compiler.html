Java Code Compiler

Sept 2018 - Dec 2018

I implemented a simple Java compiler using java which has following functionalities:

  • Scan and tokenize the input code.

  • Parse the code and make sure it is syntax correct.

  • Check variable and expression type to make sure it is grammarly correct.

  • Generate Java bytecode using ASM framework.

  • Return the output of the code.

    I used Stack, Hashtable and Tree data structures. Created and executed test cases in JUnit for unit testing of each module.
